DeepSeek Software Development Automator

This is an ongoing project working with the DeepSeek API to enhance my understanding of LLM-based tools. The goal is to develop a tool to automate the development of websites and other software projects. When completed, the user will input a description of a software project. The DeepSeek API is then used to generate a project directory structure, define all required code files, and write the code for these files. The programme then takes these instructions and generates all the required directories and files, and populates them with the necessary code, thereby creating the entire project on the user's machine.
