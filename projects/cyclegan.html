Day-to-Night Image Translation with CycleGAN and Time-Lapse Training

This project focused on possible enhancements of CycleGAN, an unsupervised learning technique, for day-to-night image translation. It also explored the possibility of incorporating time-lapse data into the training process, with the end goal of generating synthetic time-lapse sequences. The project was split into three main parts: first, the implementation of an optimised CycleGAN model for day-to-night image translation, second, experimenting with changes to the overall structure of the CycleGAN network, and finally the use of a novel network architecture to generate synthetic time-lapses.
